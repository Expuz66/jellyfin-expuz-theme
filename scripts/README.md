# Expuz Stream Scripts

Optional JavaScript helpers for the Expuz Stream theme.

- `title.js` changes the browser title to `Expuz Stream`.
- `loginimage.js` displays a selected user's profile image on the manual login screen.
- `rating.js` adds rating attributes used by `ratings.css`.
- `theme_selector.js` adds a local color preset selector.
- `10.11_dashboard_theming.js` injects the local dashboard CSS files for Jellyfin 10.11 setups that need it.
